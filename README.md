# Students Social Media Addiction Analysis

## Overview

This project performs **Exploratory Data Analysis (EDA)** on student data to understand patterns between social media usage, lifestyle, and addiction levels.

---

## Dataset

* 705 rows × 13 columns
* No missing values
* Mix of numerical and categorical data

---

## Data Processing

* Checked structure and statistics (`info`, `describe`)
* Converted categorical columns to `category` dtype

---

## Analysis Performed

* Distribution plots for numerical features
* Bar plot: mental health vs platform usage
* Boxplots for outlier detection
* Correlation heatmap

---

## Key Insights

* Higher usage → higher addiction
* Less sleep → higher addiction
* More conflicts → higher addiction
* Mental health shows a moderate correlation

---

## Tech Stack

* Python, NumPy, Pandas
* Matplotlib, Seaborn, Plotly
